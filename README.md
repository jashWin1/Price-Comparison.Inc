# Price-Comparison.inc 
A Python program I created that helps users decide which discount is better when they are buying an item.

The program will ask the user for:

1. The item that they want to buy

2. The original price

3. A percent discount 

4. A dollar discount
   
   
It then compares both of the discounts and tells the user which one saves more money.


The features of this program include:


1. User input/interaction

2. Basic math calculations

3. If/else statments

4. Some simple decision making.
   

What I learnt while coding:


1. How to use variables and the input function

2. How to be able to compare values 
   
3. How to build an interactive program 

About This Project:
